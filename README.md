# K-Connect Spotify Music Importer

Инструмент для импорта музыки из Spotify в [K-Connect](https://k-connect.ru). Позволяет переносить плейлисты, альбомы, отдельные треки и популярные треки артистов с сохранением метаданных и обложек.

Возможности

· Импорт треков, альбомов, плейлистов и топ-треков артистов из Spotify.

· Сохранение метаданных: название, исполнитель, альбом, жанр, длительность.

· Загрузка обложек альбомов.

· Многопоточная загрузка и выгрузка на K-Connect (до 10 одновременных задач).

· Авторизация в [K-Connect](https://k-connect.ru) по логину/паролю (с получением токена).

## Требования

· Python 3.7 или выше.

· Аккаунт на [K-Connect](https://k-connect.ru) с API-ключом.

· Зарегистрированное приложение на Spotify Developer для получения client_id и client_secret.

## Авторизация в K-Connect

При запуске скрипт запросит логин и пароль от [K-Connect](https://k-connect.ru). Эти данные используются для получения токена доступа.

## Использование

Запустите скрипт:

```bash
python main.py
```
